2. Theoretical Distributions

2.1. Bernoulli Distribution 伯努利分布

  1. Let YYY be a dichotomous random variable (takes one of two mutually exclusive values)

  2. Successes (= 1) occur with probability ppp and failures (= 0) occur with probability 1−p1-p1−p, for constant p∈[0,1]p \in [0,1]p∈[0,1]

  3. Notation: Y∼Bern(p)Y \sim Bern(p)Y∼Bern(p)

  4. Let Y be a dichotomous random variable representing a coin flip

    • Y=1Y = 1Y=1: heads, success
    • Y=0Y = 0Y=0: tails, fail
    • If the coin has a 60% chance to get the head/success
    • E(Y)=1⋅p+0⋅(1−p)=pE(Y) = 1\cdot p + 0\cdot(1-p) = pE(Y)=1⋅p+0⋅(1−p)=p
    • E(Y2)=12⋅(p)+02⋅(1−p)=pE(Y^2) = 1^2\cdot(p) + 0^2\cdot(1-p) = pE(Y2)=12⋅(p)+02⋅(1−p)=p
    • var(Y)=σY2=E(Y2)−E(Y)2=p−p2=p(1−p)var(Y) = \sigma^2_{Y} = E(Y^2) - E(Y)^2 = p - p^2 = p(1-p)var(Y)=σY2​=E(Y2)−E(Y)2=p−p2=p(1−p)

2.2. Binomial Distribution 二项分布

  1. Definition: If we have a sequence of nnn Bernoulli variables, each with a probability of success ppp, then the total number of successes is a binomial random variable.
    • Assumptions: fixed number of trials, independent, constant p
  2. Notation: X∼Bin(n,p)X \sim Bin(n,p)X∼Bin(n,p)
  3. Note for Combination and Permutation
    1. Combination: C(n,k)C(n,k)C(n,k) or (kn)\mathrm(^n_k)(kn​)
    2. Permutation: P(n,k)P(n,k)P(n,k)
  4. Probability Mass Function:
    1. Pr(X=x)=(xn)⋅px⋅(1−p)n−xPr(X = x) = \mathrm(^n_x) \cdot p^{x} \cdot (1-p)^{n-x}Pr(X=x)=(xn​)⋅px⋅(1−p)n−x
    2. Pr(X=x)=C(n,k)⋅px⋅(1−p)n−xPr(X = x) = C(n,k) \cdot p^{x} \cdot (1-p)^{n-x}Pr(X=x)=C(n,k)⋅px⋅(1−p)n−x
  5. Then if you flip coin for 100 times, n=100n = 100n=100, the probability to get head for k times is Pr(X=x)=C(100,k)⋅pk(1−p)100−kPr(X = x) = C(100, k) \cdot p^{k} (1-p)^{100-k}Pr(X=x)=C(100,k)⋅pk(1−p)100−k
  6. How do you calculate it in R?
    1. Calculate the probability of x successes Pr(X=x)Pr(X = x)Pr(X=x) using dbinom(x, n, p)
    2. Calculate Pr(X≤x)Pr(X \le x)Pr(X≤x) using pbinom(x, n, p)
    3. Calculate Pr(X≥x)Pr(X \ge x)Pr(X≥x) using 1 - pbinom(x - 1, n, p)
  7. Summary measures
    1. Expection E(X)=npE(X) = npE(X)=np
    2. Variance var(X)=σX2=np(1−p)var(X) = \sigma^2_X = np(1-p)var(X)=σX2​=np(1−p)
    3. Stdev σX=np(1−p)\sigma_X = \sqrt {np(1-p)}σX​=np(1−p)​
  8. How do you get those above:
    1. Consider Binomial Distribution as the sum of n times of Bernoulli Experiments
    2. When X∼Bern(p)X \sim Bern(p)X∼Bern(p)
      1. E(X)=pE(X) = pE(X)=p
      2. σX2=p(1−p)\sigma^2_X = p(1-p)σX2​=p(1−p)
    3. Then let Y∼Bin(n,p)Y \sim Bin(n,p)Y∼Bin(n,p)
      1. E(Y)=npE(Y) = npE(Y)=np
      2. σY2=nσX2=np(1−p)\sigma^2_Y = n\sigma^2_X = np(1-p)σY2​=nσX2​=np(1−p)
  9. Main take-away points from the binomial distribution:
    1. Fixed number of independent Bernoulli trials, n
    2. Constant probability of success, p (Bernoulli parameter)
    3. Interested in the total number of successes in n trials (not order)
    4. Mean: μX=np\mu_X = npμX​=np
    5. Variance: σ2=np(1−p)\sigma^2 = np(1 − p)σ2=np(1−p)
